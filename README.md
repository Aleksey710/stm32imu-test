| Supported Targets |  | ESP32-S3 |
| ----------------- |  | -------- |

esp32ahrs - проект для ESP32
docs - проект документации
docker - развертывание документации

## Поддержка WebSocket в проекте
После открытия проекта включить WebSocket в menuconfig

Открой конфиг:
idf.py menuconfig

Component config
  → HTTP Server
     → [ ] WebSocket support   ← ВКЛЮЧИТЬ

Поставь:
[*] WebSocket support
